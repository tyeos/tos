

# 常用的十六进制数：

    0x10 -> 16B
    0x40 -> 64B
    0x100 -> 256B
    0x400 -> 1024 -> 1KB (1<<10)
    0x1000 -> 4096B -> 4KB
    0x4000 -> 16KB
    0x10000 -> 64KB
    0x40000 -> 256KB
    0x100000 -> 1KB * 1KB -> 1MB (1<<20)
    0x400000 -> 4MB
    0x1000000 -> 16MB
    0x4000000 -> 64MB
    0x10000000 -> 256MB
    0x40000000 -> 1MB * 1KB = 1GB (1<<30)
    0x100000000 -> 4GB
    注：每往下一项相当于二进制后面加2个0，等同于左移2位，即乘以4。
    	每在十六进制数后加1个0，等同于左移4位，即乘以16。
